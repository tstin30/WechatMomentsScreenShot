# 项目截图预览

此目录用于存放项目的预览效果图。

## 📸 推荐的预览图

### 主要预览图
- **preview.png** - 主要的朋友圈效果图展示整体效果
  - 展示完整的配置界面
  - 包含用户信息、朋友圈内容、图片、点赞、评论等所有元素
  - 建议尺寸：800x600px，保持清晰度的同时控制文件大小

### 辅助预览图（可选）
- **config-ui.png** - 配置界面特写
- **moment-detail.png** - 朋友圈内容细节
- **multi-layout.png** - 不同布局模式对比
- **export-result.png** - 导出结果展示

## 📝 添加预览图的方法

1. **生成截图**：使用本工具生成典型的朋友圈内容
2. **截图保存**：使用系统截图工具或浏览器截图
3. **图片优化**：调整尺寸和压缩，建议PNG格式
4. **放置文件**：将图片放入此目录

## 🎨 推荐的截图内容

为了展示工具的主要功能，建议包含以下元素：

### 基础信息
- 👤 自定义头像和用户名
- 📝 朋友圈正文内容
- 📍 定位信息
- ⏰ 发表时间

### 互动功能
- 💖 多个点赞（头像模式）
- 💬 多条评论
- 🖼️ 多张图片布局

### 展示重点
- 界面的真实性还原
- 不同元素的对齐方式
- 响应式布局效果
- 导出图片的质量

## 📐 图片规范

- **格式**：PNG（推荐）
- **尺寸**：宽度不超过1200px，高度适中
- **文件大小**：单张图片不超过500KB
- **命名**：使用英文，描述性命名
