# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2025-12-26

### 🎉 主要更新
- **重构用户界面** - 优化配置面板布局，支持多列显示
- **动态高度适配** - 预览区域根据内容自动调整高度
- **评论系统重构** - 支持多条评论，实时预览更新
- **智能预览** - 所有输入项自动触发预览更新

### ✨ 新增功能
- **评论功能** - 支持评论者和评论内容输入
- **多列布局** - 用户名和头像同行显示
- **头像点击上传** - 直接点击头像区域选择图片
- **防抖优化** - 输入防抖，提升性能
- **图标优化** - 使用本地图片替代emoji符号

### 🚀 改进
- **紧凑布局** - 优化配置面板，减少空间占用
- **响应式优化** - 改进移动端和桌面端显示
- **导出优化** - 智能裁剪，无多余空白
- **交互提升** - hover效果和过渡动画

### 🐛 Bug修复
- **CORS问题** - 移除不必要的跨域属性
- **评论预览** - 修复评论输入无法触发预览的问题
- **图片加载** - 解决本地图片无法显示的问题
- **居中显示** - 评论文字水平居中对齐

### 📱 界面调整
- **操作按钮** - 移除生成预览按钮，导出按钮占满宽度
- **输入框优化** - 调整内边距和字体大小
- **状态管理** - 优化显示选项的布局
- **空状态处理** - 无评论时不显示空白区域

### ⚡ 性能优化
- **防抖机制** - 300ms防抖延迟，避免频繁更新
- **代码重构** - 模块化函数组织
- **样式优化** - 减少重复CSS规则

### 📝 文档完善
- **README重写** - 详细的使用说明和功能介绍
- **开源协议** - 添加MIT License
- **贡献指南** - 完整的贡献流程说明
- **更新日志** - 记录版本变更历史

## [1.0.0] - 2025-12-25

### ✨ 初始版本
- **基本功能** - 用户信息配置和内容编辑
- **图片上传** - 支持多图上传和布局适配
- **点赞功能** - 自动生成点赞头像和名称
- **截图导出** - 高质量PNG格式导出
- **响应式设计** - 适配不同屏幕尺寸

## 版本说明

### 版本号格式
`主版本号.次版本号.修订号`

- **主版本号** - 不兼容的API修改
- **次版本号** - 向下兼容的功能性新增
- **修订号** - 向下兼容的问题修正

### 变更类型
- 🎉 **重大更新** - 大幅功能改动或架构调整
- ✨ **新增功能** - 全新的功能特性
- 🚀 **改进优化** - 现有功能的增强
